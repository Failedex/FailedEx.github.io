What the hell is "eww" and "sway"???

sway is a compositor. I personally use swayfx, which is a fork. I like to abuse it's ipc to make sway do things it's not supposed to do.

Eww (Elkowar's wacky widget) is a software used to create widgets for your linux desktops. It's configured in a weird lisp like language I like quite a lot.
