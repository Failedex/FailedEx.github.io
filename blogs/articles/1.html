Since you're here, I'll introduce you to some facts about the website.

  • The icon for the webpage is a Petersen graph, which is a non-planar graph that does not contain K5 or K3,3.
  • Click the skull icon on the top right if I'm losing your attention (this button has been removed sorry lol)
  • The icon to the right of the skull icon is "tiling mode", commonly found in tiling window managers. After you press it, you should be able to move the windows around with left mouse button and scale the windows with the right.
  • On "floating mode", you will see a downward arrow button, which will apply gravity to the windows. The windows can be thrown around.
