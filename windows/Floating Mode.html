Floating Mode

You've just activated floating mode!

Windows can now be moved and manipulated freely with your mouse.

  • Left mouse button moves a window
  • Right mouse button resizes a window

(Note: resizes from bottom right corner)

You can press the select_window_2 button on the top right to return to tiled mode.

Unfortunately, Windows cannot be interated with in floating mode.

Gravity

Press the arrow_downward button on the top right to toggle gravity for all windows. Windows can be tossed.

This feature was copied from the SwayGravity project (made by me)
