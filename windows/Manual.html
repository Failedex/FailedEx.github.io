How do I use this??

This webpage imitates a window manager (please view this on desktop!). For people unfamiliar with how to use one, read on to learn :).

Tiling

The webpage uses DWM tiling, which uses the "master stack" layout. On the left is the "master", which is the main working window, and on the right is the "stack", which is all other windows of less importance. There is always one master window (unless there are no windows) and windows in stack can be promoted to master, which would demote the old master window to the stack.
